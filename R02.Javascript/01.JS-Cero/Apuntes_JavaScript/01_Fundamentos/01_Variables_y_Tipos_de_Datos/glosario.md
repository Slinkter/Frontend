# Glosario de Términos: Variables y Tipos de Datos

### Variable
**Definición:** En programación, una variable es un contenedor con nombre que almacena un valor. Este valor puede ser modificado y recuperado a lo largo de la ejecución del programa. En JavaScript, las variables se declaran usando `var`, `let` o `const`.

### Tipo de Dato (Data Type)
**Definición:** Un tipo de dato es una clasificación que especifica qué tipo de valor puede contener una variable y qué tipo de operaciones se pueden realizar sobre ella. Los tipos de datos determinan la naturaleza de los valores.

### Tipos de Datos Primitivos (Primitive Data Types)
**Definición:** Son los tipos de datos más básicos y fundamentales en JavaScript. Son inmutables, lo que significa que su valor no puede ser alterado una vez creado. Los seis tipos de datos primitivos son:
- **`string`**: Una secuencia de caracteres, como `"Hola, mundo"`.
- **`number`**: Un valor numérico, que puede ser entero o de punto flotante.
- **`boolean`**: Un valor lógico, que puede ser `true` o `false`.
- **`null`**: Representa la ausencia intencional de un valor de objeto.
- **`undefined`**: Indica que una variable ha sido declarada pero no se le ha asignado un valor.
- **`symbol`**: Un tipo de dato único e inmutable, a menudo utilizado como clave en objetos.

### Objeto (Object)
**Definición:** Un objeto es una estructura de datos compleja que permite almacenar colecciones de datos y funcionalidades más complejas. Consiste en un conjunto de pares clave-valor, donde las claves son strings (o Symbols) y los valores pueden ser cualquier tipo de dato, incluyendo otros objetos.

### Declaración (Declaration)
**Definición:** La declaración es el proceso de crear una nueva variable en el código. En JavaScript, esto se hace usando las palabras clave `var`, `let` o `const` seguidas del nombre de la variable.
- **Ejemplo:** `let edad;`

### Inicialización (Initialization)
**Definición:** La inicialización es el acto de asignar un valor inicial a una variable en el momento de su declaración.
- **Ejemplo:** `let edad = 25;`

### Mutabilidad (Mutability)
**Definición:** La mutabilidad se refiere a la capacidad de un valor para ser modificado después de su creación. En JavaScript, los objetos y los arrays son mutables, mientras que los tipos de datos primitivos son inmutables.
