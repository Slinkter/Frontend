# Glosario: Clases

## H

### Herencia Prototípica

Método por el cual un objeto puede heredar propiedades y métodos de otro objeto. A diferencia de la herencia de clases clásica (donde se copia la estructura), en JS es un enlace vivo hacia el objeto prototipo.

## I

### Instancia

Un objeto concreto creado a partir de una clase o función constructora.

## M

### Mixin

Una clase o objeto que ofrece cierta funcionalidad para ser heredada por una subclase, pero que no está destinada a ser instanciada por sí sola. Permite simular herencia múltiple.

## P

### Polimorfismo

Capacidad de objetos de diferentes clases de responder al mismo mensaje (método) de maneras distintas.

## S

### Sugar Syntax (Azúcar Sintáctico)

Sintaxis diseñada para hacer el código más fácil de leer o expresar, sin añadir nueva funcionalidad al lenguaje. Las `class` en JS son azúcar sobre los prototipos.

### Super

Palabra clave usada para llamar funciones del padre de un objeto.
