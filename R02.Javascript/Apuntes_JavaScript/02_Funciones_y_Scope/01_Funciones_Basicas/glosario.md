# Glosario: Funciones

## C

### Callback

Una función pasada como argumento a otra función, para ser invocada ("llamada de vuelta") en un momento posterior.

### Ciudadano de Primera Clase (First-Class Citizen)

Entidad que soporta todas las operaciones generalmente disponibles para otras entidades (ser asignada a variables, pasada como argumento, retornada).

## H

### Higher Order Function (Función de Orden Superior)

Una función que cumple al menos una de las siguientes condiciones:

1. Toma una o más funciones como argumentos.
2. Retorna una función como resultado.

## I

### IIFE

Siglas de "Immediately Invoked Function Expression". Una expresión de función que se ejecuta tan pronto como se define.

## P

## Parámetro Rest (`...args`)

Sintaxis que permite a una función aceptar un número indefinido de argumentos como un array. `function (a, b, ...moreArgs) {}`.
