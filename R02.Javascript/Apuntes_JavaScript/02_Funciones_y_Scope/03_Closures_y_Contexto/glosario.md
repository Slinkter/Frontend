# Glosario: Closures y this

## B

### Binding (Enlace)

La asociación de un identificador a un valor o, en el contexto de `this`, la asociación de la función con el objeto que la invoca.

## C

### Closure (Clausura/Cierre)

Una función que tiene acceso a variables de su ámbito léxico externo, incluso después de que la función externa haya retornado.

### Context (Contexto)

A menudo se refiere al valor de `this` dentro de una función. No debe confundirse con Scope.

## E

### Encapsulamiento

Principio de la POO que agrupa datos y métodos que operan sobre esos datos en una sola unidad, restringiendo el acceso directo a algunos de los componentes del objeto. En JS funcional, los Closures permiten encapsulamiento.

## M

### Module Pattern (Patrón Módulo)

Patrón de diseño que utiliza closures (generalmente con IIFEs) para crear métodos públicos y privados, protegiendo variables del scope global.
