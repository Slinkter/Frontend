# Glosario: Web APIs

## D

### DOM (Document Object Model)

Interfaz de programación para documentos HTML y XML. Representa la página para que los programas puedan cambiar la estructura, estilo y contenido del documento. El DOM representa el documento como nodos y objetos.

## J

### JSON (JavaScript Object Notation)

Formato ligero de intercambio de datos. Es fácil de leer y escribir para humanos, y fácil de parsear y generar para máquinas. `JSON.stringify` convierte objeto a string, `JSON.parse` string a objeto.

## S

### Serialización

El proceso de convertir un objeto en memoria (como un objeto JS) en una secuencia de bytes o caracteres (como un string JSON) para que pueda ser almacenado o transmitido.

## X

### XSS (Cross-Site Scripting)

Vulnerabilidad de seguridad donde un atacante inyecta scripts maliciosos en páginas web visitadas por otros usuarios. Usar `innerHTML` con datos no confiables es una causa común.
