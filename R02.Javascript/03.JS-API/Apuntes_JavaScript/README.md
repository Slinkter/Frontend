# Apuntes de JavaScript: Curso de Nivel Postgrado

Bienvenido a este curso de JavaScript, diseñado para consolidar y expandir su conocimiento desde los fundamentos hasta las aplicaciones avanzadas. Este material ha sido meticulosamente reestructurado a partir de sus notas de estudio, transformándolas en un recurso académico profesional y coherente.

El curso está organizado en cinco módulos principales, cada uno profundizando en aspectos críticos del desarrollo con JavaScript.

---

## Estructura del Curso

Aquí encontrará un desglose de los módulos y los temas que cubriremos, con enlaces directos a cada lección.

### **Módulo 01: Fundamentos Nucleares de JavaScript**
Este módulo sienta las bases, explorando los pilares esenciales del lenguaje.
- [**Variables, Alcance (Scope) y Memoria**](./01_Fundamentos/variables_y_scope.md)
- [**Tipos de Datos y Coerción**](./01_Fundamentos/tipos_de_datos.md)
- [**Operadores y Coerción**](./01_Fundamentos/operadores_y_coercion.md)
- [**Estructuras de Control y Flujo**](./01_Fundamentos/estructuras_de_control.md)

### **Módulo 02: Abstracción y Estructuras de Datos**
Nos adentraremos en cómo JavaScript permite organizar y manipular información y lógica compleja.
- [**El Rol de las Funciones**](./02_Funciones_y_Estructuras/funciones.md)
- [**Manejo de Colecciones: Arrays y sus Métodos**](./02_Funciones_y_Estructuras/arrays_y_metodos.md)
- [**El Objeto en Profundidad y JSON**](./02_Funciones_y_Estructuras/objetos_avanzados.md)

### **Módulo 03: El Modelo Asíncrono de JavaScript**
Comprender la asincronía es fundamental para el desarrollo web moderno. Este módulo desmitifica cómo JavaScript maneja operaciones que toman tiempo.
- [**El Corazón de JS: El Event Loop**](./03_Asincronia/event_loop.md)
- [**De Callbacks a Promesas: Evolución de la Asincronía**](./03_Asincronia/callbacks_y_promises.md)
- [**`async/await`: La Sintaxis Moderna para un Código Legible**](./03_Asincronia/async_await.md)
- [**Estrategias de Manejo de Errores Asíncronos**](./03_Asincronia/manejo_errores_asincronos.md)

### **Módulo 04: El Navegador como Entorno de Ejecución**
Exploraremos la interacción entre JavaScript y el entorno del navegador, vital para cualquier aplicación web frontend.
- [**APIs del Navegador: DOM y BOM**](./04_Web_APIs_y_DOM/introduccion_web_apis.md)
- [**Manipulación Avanzada del DOM**](./04_Web_APIs_y_DOM/manipulacion_dom.md)
- [**Gestión de Eventos del Usuario**](./04_Web_APIs_y_DOM/eventos_dom.md)
- [**Creando una Single Page Application (SPA) con `location.hash`**](./04_Web_APIs_y_DOM/navegacion_spa.md)
- [**Persistencia de Estado con `localStorage`**](./04_Web_APIs_y_DOM/local_storage.md)

### **Módulo 05: Comunicación, Performance y Patrones**
Este módulo cubre técnicas avanzadas para la comunicación con servidores, la optimización del rendimiento y la aplicación de patrones de diseño.
- [**`Fetch` API: El Estándar para Peticiones HTTP**](./05_Comunicacion_y_Performance/fetch_api.md)
- [**Abstracciones con Librerías Externas: El caso de `Axios`**](./05_Comunicacion_y_Performance/axios_libreria.md)
- [**Patrones de Interacción con APIs REST (Autenticación, Métodos)**](./05_Comunicacion_y_Performance/interaccion_apis_rest.md)
- [**Optimización del Renderizado: Lazy Loading y `IntersectionObserver`**](./05_Comunicacion_y_Performance/optimizacion_performance.md)
- [**Patrón de Diseño: Scroll Infinito para Experiencia de Usuario Fluida**](./05_Comunicacion_y_Performance/scroll_infinito.md)

**Metodología:**

Cada tema se presentará con un enfoque académico, incluyendo:
- **Clase:** Explicación teórica profunda y profesional.
- **Ejemplos:** Fragmentos de código comentados y contextualizados, a menudo derivados y mejorados de sus propias notas.
- **Ejercicios:** Retos prácticos para afianzar el conocimiento, con soluciones razonadas.
- **Glosario:** Definiciones técnicas precisas de los términos clave.

¡Empecemos a construir este conocimiento!
